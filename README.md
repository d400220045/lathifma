### 🚀 Proyek Unggulan

#### 🌱 Sistem Irigasi Otomatis Berbasis Energi Surya dan IoT
Sistem irigasi otomatis menggunakan **ESP32**, sensor kelembapan tanah, dan panel surya, dengan monitoring kontrol energi, daya, dan kelembapan tanah secara real-time melalui platform IoT.
`ESP32` `IoT` `PLTS` `Sensor Kelembapan Tanah`

#### ⚡ Monitoring Daya SPKLU Berbasis PLTS dan IoT
Sistem monitoring energi menggunakan **ESP32, PZEM, dan sensor baterai** untuk memantau tegangan, arus, daya, energi, dan SOC baterai secara real-time. Termasuk pengujian, kalibrasi, dan troubleshooting hingga sistem stabil.
`ESP32` `PZEM` `Modbus RS485` `IoT` `Battery Monitoring`

#### 🏆 Sistem Lomba Cerdas Cermat Berbasis Arduino
Perangkat tombol cepat berbasis **Arduino** untuk kegiatan MPR RI, dirancang agar berjalan stabil selama pelaksanaan acara.
`Arduino` `Embedded System`
